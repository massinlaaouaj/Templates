git init				-->Create a new git repository.

git remote add origin URL_of_repo	-->Save the URL of the github or gitlab repository in the "origin" tag. EYE this URL is that of the repository, not that of "Clone or download", but that of the browser bar.

git add				-->To add to the Statge phase, and upload it later, the file or the entire directory.

git commit -m "version"		-->We do a COMMIT as you are going to push it.

git push origin master		-->To upload the changes made to the COMMIT to the repository.
